# Learn Go from Scratch (14 Days)

This package is designed for complete beginners to learn Go in a structured, concise, and practice-focused way.

## Package Contents

- `curriculum/14-days.md` -> daily learning plan
- `curriculum/gin-14-days.md` -> 14-day Gin framework curriculum
- `exercises/questions-and-answers.md` -> daily exercises + sample answers
- `projects/mini-cli/main.go` -> sample mini CLI project

## How to Use

1. Install Go on Windows:
   - `winget install GoLang.Go`
   - verify: `go version`
2. Open this folder in a terminal.
3. To run the mini project:
   - `cd projects/mini-cli`
   - `go run .`

## Daily Learning Pattern (60-90 minutes)

- 20 minutes: read today's topic
- 30 minutes: solve exercises without looking at answers
- 10 minutes: compare with sample answers
- 10-30 minutes: modify and improve the program yourself

## Goals After 14 Days

- Understand basic Go syntax
- Use functions, structs, methods, and simple interfaces
- Handle errors idiomatically (`if err != nil`)
- Understand goroutines and channels at a basic level
- Build a small CLI application
